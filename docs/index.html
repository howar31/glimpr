Made for the exact pixel為精確到像素而生

Pixel loupe像素放大鏡

One grid cell is one screen pixel. The loupe names the pixel you are on; arrow keys nudge by exactly one. 一格就是一個螢幕像素。放大鏡指名你所在的像素；方向鍵每次精準微調一格。

Annotate on a frozen frame在凍結畫面上標註

Shapes, arrows, pen, highlighter, text, blur, spotlight and step badges — across every display at once. 形狀、箭頭、畫筆、螢光筆、文字、模糊、聚光燈與步驟標記 — 所有螢幕同時進行。

Record the screen螢幕錄影

Pin to screen釘選到螢幕

Install安裝

macOS 14+

  1. Download the macOS .dmg 下載 macOS 版 .dmg
  2. Drag Glimpr into Applications 把 Glimpr 拖進「應用程式」
  3. Grant Screen Recording on first launch — the app is signed and notarized 首次啟動時授權「螢幕錄製」— app 已簽章並公證

Windows 10 (1903+) / 11

  1. Run the downloaded Glimpr-Setup installer 執行下載的 Glimpr-Setup 安裝程式
  2. SmartScreen: More info → Run anyway — expected while the release is unsigned SmartScreen：其他資訊 → 仍要執行 — 未簽章版本的預期行為
  3. Or unzip the portable build anywhere 或將可攜版解壓到任意位置
